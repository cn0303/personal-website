# Sketch 001: Home / Hero Direction

## Design Question
What is the site's core visual language? This sketch puts three dramatically
different directions side by side, all built from the same intake references
(20n, naklecha, provigen, eka, latentworlds) and the same font palette
(Merriweather serif × Satoshi sans × JetBrains Mono). The winner sets type,
palette, mood, and motion posture for everything downstream.

## How to View
open .planning/sketches/001-home-hero-direction/index.html

Use the **top bar** to switch variants and the **bottom-right toolbar** to
swap viewport (Full / 768 / 375) — mobile is a first-class check here.

## Variants
- **A: Light Editorial** — warm off-white, large Merriweather serif headline with sans lede, a wireframe robot arm on a blueprint grid whose **gripper follows your cursor** (the latentworlds gimmick), dark pill CTAs, and a "What I'm betting on" peek. Calm, confident, editorial.
- **B: Dark Cinematic-Technical** — near-black with a faint blueprint grid + full-bleed media zone (eka), bold Satoshi headline with an *italic-serif interleave* (provigen's "Better Decisions *every Experiment*"), mono kicker labels, and provigen-style technical panels.
- **C: Manifesto Minimal** — stark, whitespace-heavy, a distinctive abstract monogram mark, and a monospace thesis statement centered on the page (20n). Restrained to the point of being a statement.

## What to Look For
- **Light vs dark** — which mood feels more *you*?
- **Type interleave** — does the serif×sans (×italic) mix read as crafted or busy?
- **The cursor robot (A)** — does that dynamic moment delight, or distract?
- **Mobile** — switch to 375px on each variant; does it still feel intentional?
- **Voice fit** — ignore the placeholder words; does the *structure* hold your story?

## Outcome — Variant B selected (Dark Cinematic-Technical)
Owner picked **B** decisively ("provigen + eka combined really well"). Rejected A
(light editorial) and C (manifesto minimal). **The cursor-following robot gimmick
was explicitly rejected** (janky linkage + too literal a copy of latentworlds) and
removed — Variant A's arm is now static and preserved only as a rejected reference.

**Carry-forward decisions:**
- Direction = **dark cinematic-technical**: near-black, faint blueprint grid, full-bleed media zone, mono technical labels, provigen-style panels.
- Type interleave (bold Satoshi headline × italic Merriweather emphasis × mono labels) = approved.
- **Open issue → sketch 002:** the wordmark/name treatment is too plain and flat; needs a larger, more distinctive personal-site identity treatment.
- Media zones will use owner's real footage/stills (optionally AI/diffusion-retouched to match the dark grade) — asset task for later phases.
